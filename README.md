## 概述

用于在 Render 上部署 [cloudreve](https://cloudreve.org/)

本镜像基于 debian:stable-slim

Aria2地址：http://127.0.0.1:6800

Aria2秘钥：123

数据库免费注册推荐: [SQLPub](http://mysql.sqlpub.com/)
